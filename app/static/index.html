⏰ Chronos Context

AI-powered project analysis & context generation API for LLM integration.

Projects

GET /projects

List all registered projects.

POST /projects

Register a new project. Body: {"name", "path"}.

{
  "name": "my-app",
  "path": "/path/to/project"
}
GET /projects/{project_id}

Get project details.

DELETE /projects/{project_id}

Delete project and all its data.

Snapshots

GET /projects/{project_id}/snapshots

List all snapshots for a project.

GET /snapshots/{snapshot_id}

Get details of a specific snapshot.

Analysis

GET /projects/{project_id}/context

Get context JSON for a project (auto-analyzes if no snapshot).

POST /analyze

Run analysis. Query param: ?use_cache=true.

{
  "project_path": "/path/to/project"
}

Export Formats

POST /projects/{project_id}/export

Export project context. Body: {"format", "max_tokens", ...}. Formats: txt, md, gpt, claude.

{
  "format": "gpt",
  "include_imports": true,
  "depth": 1
}

Settings

GET /projects/{project_id}/settings

Get project settings.

PUT /projects/{project_id}/settings

Update project settings.

Cache

POST /invalidate-cache

Invalidate analysis cache for a project.

{
  "project_id": 1
}
