# Chronos Context - План разработки проекта

## 📋 Обзор проекта

**Chronos Context** — веб-сервис для подготовки, хранения и выдачи сжатого контекста Python-проектов под LLM-кодинг. Экономия 85-95% токенов.

---

## ✅ Текущее состояние (MVP)

| Компонент | Статус | Описание |
|-----------|--------|----------|
| Core Engine | ✅ Базовый | `core/engine.py` — парсер Python (ast), компрессор, счётчик токенов (tiktoken) |
| Backend API | ✅ Базовый | FastAPI: `/projects/`, `/analyze/`, `/snapshots/` |
| База данных | ✅ Базовый | SQLModel + SQLite, модели Project и ProjectSnapshot |
| Web UI | ✅ Базовый | HTML/JS фронтенд с метриками и просмотром структур |
| Сжатие | ✅ Рабочий | Python-only, ~85-95% экономия токенов |
